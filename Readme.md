## Music Streaming Backend

This backend provides APIs for a Music Streaming Web Application built using Node.js, Express, and MongoDB.
It manages user authentication, songs, playlists, likes, and search functionality.

## Features

User authentication with JWT

Create and manage playlists

Add or remove songs from playlists

Like and unlike songs

Search songs

Password reset via email

Tech Stack

Node.js

Express.js

MongoDB

Mongoose

JWT Authentication

bcrypt

Nodemailer

Run the Server

Install dependencies:

npm install

## Start the server:

npm run dev

The server will run on:

http://localhost:4000
